## Project Description
A full-stack AI engineering prototype demonstrating core AI engineering concepts hands-on. The platform aggregates job listings from multiple sources and scores them against a resume using GPT-4o-mini, surfacing top matches via a Telegram bot and daily email digest. A RAG chatbot lets recruiters query the resume using natural language, powered by OpenAI embeddings stored in Supabase pgvector. The system includes an MCP server, LLM tracing with Langfuse, prompt caching, and automated daily pipelines — all running on Vercel.
